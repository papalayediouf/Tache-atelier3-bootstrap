le développement web

Competences

HTML

Langage de balisage conçu pour représenter les pages web.

CSS

Un langage informatique qui décrit la présentation des documents HTML et XML.

BOOTSRAP

Bootstrap est une collection d'outils utiles à la création du design de sites et d'applications web.

Front-End

Le développement web frontal (aussi appelé front-end en anglais) correspond aux productions HTML, CSS et JavaScript d'une page internet ou d'une application qu'un utilisateur peut voir et avec lesquelles il peut interagir directement.

Back-End

Le backend est responsable du stockage et de la récupération des données à partir de la base de données.
